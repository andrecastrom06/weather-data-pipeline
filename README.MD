# Weather Data Pipeline

## O que foi utilizado no projeto?
- Apache Airflow
- Docker
- Arquitetura Medalhão
- SQL Alchemy
- Requests
- API (openweathermap)
- DAG
- Orquestração de Pipeline
- Postgresql

## Como acessar o banco?
- docker exec -it etl-postgres-1 psql -U airflow -d airflow
- Pode fazer a consulta
